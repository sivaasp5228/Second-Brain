# GAPPY AI – Second Brain

GAPPY AI is a premium, enterprise-grade AI Second Brain SaaS application designed to transform knowledge into action. It connects your notes, documents, PDFs, links, voice notes, and captured ideas into one intelligent, interactive workspace.

Designed with a high-end dark theme (`#0B1220`), vibrant blue-to-purple gradients, glassmorphism, responsive components, and fluid animations.

---

## 🚀 Key Features

| Screen | Description |
|---|---|
| **1. Landing / Hero** | TechCrunch-grade entrance featuring an animated interactive neural network particle canvas. |
| **2. Dashboard** | Central command center with welcome counters, search bar, and recent second-brain activity. |
| **3. Upload Knowledge** | Drag-and-drop file uploader supporting PDF, DOCX, links, audio, and URL parsing. |
| **4. AI Chat** | ChatGPT-style messaging with real-time response streaming, details for model thought processes, and action items. |
| **5. Knowledge Library** | Filterable library grid integrated with the official Lemma datastore table. |
| **6. Insights** | Recharts area/bar charts displaying knowledge growth and key topics over a timeline. |
| **7. Decision Assistant** | AI recommendation analyzer with confidence gauges and side-by-side comparison tables. |
| **8. Tasks (Kanban)** | Board for managing tasks generated by AI conversations with column flow. |
| **9. Knowledge Graph** | Interactive canvas-rendered 2D force-directed network showing node relationships. |
| **10. Settings** | Settings panel to configure profile, notification preferences, and API keys. |

---

## 🛠️ Technology Stack

- **Core**: React 18 + TypeScript + Vite 8
- **Styling**: Tailwind CSS v4 + Custom Glassmorphism System
- **Animation**: Framer Motion (Transitions, Counters, Hover Micro-animations)
- **Database/Agent Integration**: Official `lemma-sdk` + `@lemma-sdk/react`
- **Charts**: Recharts
- **Icons**: Lucide React
- **Client Cache**: `@tanstack/react-query`

## ⚙️ Local Development & Setup

### 1. Environment Variables Configuration

To connect GAPPY AI to the Lemma Cloud workspace and authenticate users, create a `.env` file at the root of the project:

```env
VITE_LEMMA_API_URL=/api
VITE_LEMMA_AUTH_URL=https://lemma.work/auth
VITE_LEMMA_POD_ID=019f217a-74c3-71bd-9999-9644cd5bd501
```

*Note: In development, the client relies on a relative path `/api` proxy configured in `vite.config.ts` to bypass CORS and securely carry cookies to the Lemma endpoints.*

### 2. Install Dependencies

```bash
npm install
```

### 3. Run the Development Server

```bash
npm run dev
```

Your app will start on [http://localhost:5174/](http://localhost:5174/).
